# NatDevice-ZXing.NET Integration
This is an example project which integrates the NatDevice media device API with ZXing.NET for realtime barcode detection.

*ILLUSTRATION*.

## Requirements
- Unity 2019.2+
- NatDevice 1.0.3+

## Setup Instructions
- Download and open the project in Unity.
- Import [NatDevice API from the Asset Store](https://assetstore.unity.com/packages/tools/integration/natdevice-media-device-api-162053).
- Set your build target to iOS or Android.
- Build and run!
